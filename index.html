YoomJS - MVC Framework for NodeJS

YoomJS is a next generation web framework for NodeJS built on top of KoaJS. YoomJS mimics the MVC pattern of frameworks like Ruby on Rails which makes writing custom, enterprise-grade NodeJS applications fast and enjoyable.

Installation

Dependencies

Make sure that NodeJS version 0.11.13 or later is installed. You can easily installing a fresh version of NodeJS using the Node Version Manager.

You also need Bower and Gulp commands available from the command-line.

Yoom

Install the package.

npm install -g yoom

After the installation the yoom command-line script will be available. Run yoom --help to see the list of available commands.

Create New Project

Use the yoom command to create a new application directory structure.

yoom new {project-path}

From {project-path} start the application server.

yoom start

You can also run open the application in your default browser.

yoom open

The project is ready and you can start building your next generation web application, API, or website.
