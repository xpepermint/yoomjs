YoomJS - MVC Framework for NodeJS

YoomJS is a next generation web framework for NodeJS built on top of KoaJS. YoomJS mimics the MVC pattern of frameworks like Ruby on Rails which makes writing custom, enterprise-grade NodeJS applications fast and enjoyable.

Installation

Before you start: Make sure that NodeJS version 0.11.13 or later is installed. If not we recommend installing a fresh version of NodeJS using the Node Version Manager.

Install the NPM package.

npm install -g yoom

Create a new project.

yoom new {myproject}

Start the application server.

cd {myproject}
yoom start

Open your favourite browser and navigate to http://localhost:3000. The project is ready and you can start building your next generation web application or API.
